# 请假管理系统 (Leave Management System)

一个基于纯前端技术构建的现代化请假管理系统演示站点，提供完整的请假申请、审批、记录管理功能。

## 🎯 项目特点

### 技术栈
- **前端框架**: 纯HTML5 + CSS3 + JavaScript (ES6+)
- **路由系统**: 自定义Hash路由
- **数据存储**: LocalStorage + 模拟数据文件
- **UI设计**: 响应式布局，适配移动端
- **依赖**: Lucide图标库 (CDN)

### 核心特性
- ✅ **完整的请假流程**：申请 → 审批 → 记录管理
- ✅ **智能假期余额计算**：自动计算剩余假期天数
- ✅ **多级审批**：支持两级审批流程
- ✅ **状态管理**：完整的请假状态追踪（Pending、Approved、Rejected、Cancelled）
- ✅ **用户权限控制**：基于角色的权限管理
- ✅ **实时日期计算**：自动计算工作日天数
- ✅ **响应式设计**：适配PC端和移动端
- ✅ **模拟数据支持**：内置完整的测试数据

## 📋 功能模块

### 1. 登录系统
- 支持用户名密码登录
- 自动验证用户状态（离职用户不可登录）
- 登录时自动加载最新模拟数据

### 2. 首页仪表板
- 显示用户基本信息
- 实时展示假期余额
- 快速访问请假申请

### 3. 请假申请
- 支持多种请假类型（年假、病假、事假、产假）

- 实时验证请假天数是否超过剩余额度
- 支持上传附件（病假、产假）
- 自动填充员工信息和当前日期

### 4. 我的记录
- 查看所有请假记录
- 按状态筛选记录
- 编辑审批中的请假单
- 查看详细申请信息

### 5. 审批中心
- 查看待审批的请假申请
- 支持审批/拒绝操作
- 查看审批历史

### 6. 系统管理
- 用户管理
- 请假类型管理
- 假期额度管理

## 📊 数据模型

### 用户数据 (users.json)
- 包含用户基本信息、角色、状态等
- 支持在职/离职状态管理

### 请假类型 (leaveTypes.json)
- 年假、病假、事假、产假
- 每种类型有独立的颜色标识
- 支持是否需要附件

### 假期额度 (entitlements.json)

- 自动计算已使用天数

### 请假记录 (leaves.json)
- 完整的请假流程状态
- 支持多级审批
- 详细的申请信息

## 📝 使用指南

### 请假申请流程
1. 登录系统
2. 点击「申请」按钮进入申请页面
3. 选择请假类型和日期范围
4. 系统自动计算请假天数
5. 填写请假原因
6. 上传必要的附件（如病假证明）
7. 提交申请

### 审批流程
1. 经理登录系统
2. 进入「审批」页面
3. 查看待审批的请假申请
4. 点击「审批」按钮进行处理
5. 选择批准或拒绝
6. 填写审批意见

### 记录查询
1. 进入「记录」页面
2. 查看所有请假记录
3. 点击记录查看详细信息
4. 审批中的记录可以编辑修改
